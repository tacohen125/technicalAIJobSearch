# Interview Preparation

Use this guide to help the user prepare for interviews at target companies.

## Generating Practice Questions

Tailor questions to three categories based on the job description:

1. **Behavioral questions**
   - Map each job requirement to a likely behavioral question
   - Focus on leadership, conflict resolution, cross-functional collaboration, and ambiguity
   - Example: If the role requires "stakeholder management," expect "Tell me about a time you had to align competing stakeholder priorities"

2. **Technical questions**
   - Identify tools, methodologies, and domain knowledge from the job posting
   - For TPM roles: program lifecycle, risk management, trade-off decisions, technical architecture discussions
   - For engineering roles: systems design, integration patterns, debugging approaches

3. **Situational/case questions**
   - "How would you approach..." scenarios drawn from the role's responsibilities
   - Prioritization exercises, resource allocation, launch/no-launch decisions

## Crafting STAR Responses

Use the user's experience from `references/user_profile.md` and the baseline resume to build responses:

- **Situation**: Set the context concisely (1-2 sentences)
- **Task**: Clarify the user's specific responsibility
- **Action**: Detail what the user did — use action verbs from the job posting
- **Result**: Quantify the outcome with specific metrics (team sizes, dollar amounts, timelines, percentage improvements)

Prepare 5-7 STAR stories that can be adapted across multiple questions. Each story should map to 2-3 common behavioral themes.

## Questions to Ask Interviewers

Generate role-specific questions that demonstrate:
- Understanding of the team's current challenges
- Genuine interest in the company's direction (use findings from company research)
- Thoughtfulness about how the user would contribute

Avoid generic questions ("What's the culture like?"). Prefer specific ones ("I saw the team recently launched X — how has that changed your priorities for this role?").

## Interview Format Preparation

If the interview format is known, tailor preparation accordingly:
- **Phone screen**: Concise elevator pitch, top 3 relevant experiences, salary expectations
- **Hiring manager**: Deeper STAR stories, management philosophy, team dynamics questions
- **Panel**: Adapt tone for mixed audiences (technical + non-technical)
- **Case study/presentation**: Structure, time management, anticipating questions
